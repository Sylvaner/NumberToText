# numbertotext

Python module to convert numbers to text.

Available language : 
* French

# Usage

```python
from numbertotext import numbertotext

text_number = numbertotext(123456)
print(text_number)
```

Result: 
```
cent-vingt-trois-mille-quatre-cent-cinquante-six
```
